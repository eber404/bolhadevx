Você é um designer de UX/UI responsável por criar a interface de uma SPA em React que imita um aplicativo do Windows 95. O objetivo é criar uma experiência retrô, simples, divertida e totalmente temática para um gerador de tweets satíricos da “bolha dev”.

Desenvolva os wireframes, fluxos e especificações de UI considerando:

- A aplicação possui uma única janela central estilo Windows 95.
- Toda a UI segue padrões retrô: botões 3D com bevel, inputs clássicos, MS Sans Serif, janelas arrastáveis, checkboxes Win95, dropdowns retrô e bordas afundadas.
- O usuário seleciona:
  - Categorias e tópicos de críticas.
  - Formato estrutural do tweet (ex: hot take, thread fake, opinião polêmica).
  - Gatilhos de engajamento (sarcasmo, ironia, pergunta final, emoji obrigatório).
- Uma prévia dinâmica do tweet aparece dentro da janela principal.
- Ao clicar em “Generate Hot Take”, surge uma barra de progresso fake no estilo Win95 com mensagens cômicas.
- O resultado é exibido em um popup retrô com:
  - Visualização estilizada do tweet.
  - Um botão “Copy” (sem login ou publicação).
- A experiência deve ser intuitiva, engraçada e totalmente alinhada com o tema retrô.

Entregue um documento contendo:

1. Mapa da tela e layout principal.
2. Lista dos componentes visuais necessários.
3. Estados de cada componente (idle, hover, pressed, disabled).
4. Especificações de comportamento e interações.
5. Recomendações de microinterações e animações retrô.
6. Paleta e estilo visual.
7. Considerações de acessibilidade dentro do estilo retrô.
